# Plant Management System

Batch serial number management for production plants. Create batches, auto-generate serial numbers, and activate via API.

## Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/mohit-ambani/plant-management)

Click the button above to deploy. Render will automatically set up the web app + PostgreSQL database.

## Local Development

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (separate terminal)
cd frontend && npm install && npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:3001
